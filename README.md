## Análise Exploratória de Dados
#### Utilizando Python + (Pandas, Matplotlib, Seaborn)

Este repositório contém os arquivos utilizados para realizar o trabalho da disciplina de Metodlogia Científica e Técnicas de Experimentação para Ciência da Computação do Mestrado em Ciência da Computação da Unioeste.

Contém, especificamente:
- O arquivo com os códigos Python para realizar a análise exploratória dos dados fornecidos;
- O _dataset_ milsa (em formato csv) que foi utilizado como fonte de dados nos códigos.

Obs.: Parte dos códigos foram fornecidos pela docente da disciplina (profª Simone Aparecida Miloca), os demais foram desenvolvidos com base nos códigos fornecidos ou fontes terciárias.
